# 追想の窓辺

![「追想の窓辺」のプレビュー画像。人形「スマルティーナ」の顔のアップ](./thumbnail.png)

## これは何？

伺か用ゴースト「追想の窓辺」の公開レポジトリです。

ダウンロードは[Releasesの最新版](https://github.com/tukinami/doll_at_the_bay_window/releases/latest)から`doll_at_the_bay_window.nar`をどうぞ。

## ゴースト概要

ボロボロの窓辺に佇む人形「スマルティーナ」を眺めるゴーストです。
時間が経つと、度々小鳥が来たり、手紙が風にとばされてきたりします。

masterシェルが大きいので、適宜ベースウェアの設定で縮小することをおすすめします。
